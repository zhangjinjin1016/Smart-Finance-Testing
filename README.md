# 智慧理财 - 功能测试项目

## 项目背景

智慧理财是一个P2P借贷平台，有借款人、投资人、平台审核员三个角色。核心业务包括额度申请、借款发布、投资、充值提现等。这个项目是我找来做的Web手工功能测试实战。

## 测试范围

- 注册、登录、开户
- 额度申请（额度类型：信用/担保/抵押/流转，金额1~1,000,000）
- 额度审核（通过金额 ≤ 申请金额）
- 借款申请（信用标：金额100~200,000且为50倍数，年利率3%~10%）
- 借款业务流程（借款人 → 审核员 → 发布标 → 投资列表）
- 投资（投资金额10倍数，余额不足跳转充值）

## 测试成果

- 设计测试用例86条（正向、逆向、边界值），计划执行70条（P0/P1/P2优先级），实际执行70条，通过65条，通过率92.9%
- 发现5个缺陷（严重1个、一般3个、轻微1个），已修复4个，修复率80%

**典型缺陷**：
- 借款投资密码输入5位纯数字，预期提示6~16位字母数字组合，实际发布借款成功
- 筹标期限设为256天（需求最大值），无错误提示且申请失败
- 项目名称输入25个汉字提示超长，但需求要求50字符以内
